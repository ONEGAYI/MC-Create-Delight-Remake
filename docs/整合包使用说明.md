# **「齿轮盛宴·魔改版」使用说明**

*~~其实也没有什么魔改的地方~~*
## **第零步** 针对纯萌新

**如果你是第一次玩 MC，或者没有用过 MC 启动器 HMCL:**
1. 准备 Java 环境
   - 推荐安装 Java 17 或 21
   - [下载地址](https://www.oracle.com/cn/java/technologies/downloads/#java21) 或自行搜索 Java
   - (跟着其他教程) 安装完毕，`Win`+`R`键打开`运行`，输入`cmd`，然后弹窗中输入`java --version`出现类似下面的打印就说明安装成功了
    ```log
    java 21.0.8 2025-07-15 LTS
    Java(TM) SE Runtime Environment (build 21.0.8+12-LTS-250)
    Java HotSpot(TM) 64-Bit Server VM (build 21.0.8+12-LTS-250, mixed mode, sharing)
    ```

2. 准备 HMCL 环境

    !!! note 游戏启动器 HMCL 的介绍
        **关于 HMCL 的介绍**
        - 如果你不了解 HMCL，可以理解为登陆器
        - [下载地址](https://hmcl.huangyuhui.net/download/) 或者浏览器搜索 HMCL
        - 首次使用你需要取一个角色名字 - 在主界面左上角
            ![HMCL 账户](<assets/pics/整合包使用说明/HMCL 账户.png>)
            - 取名推荐优先 LittleSkin 外置服务器认证 (需要注册，稍微麻烦，但有皮肤) <u>*如果没有该选项，检查系统地区是否为大陆*</u>
            - 怕麻烦直接离线模式
        
        !!! warning 取名注意
            取一个 **<mark>人类</mark>** 的名字

            你在一个服务器的**资产与你的名字强绑定** (尤其在非正版认证的服务器中)，所以改名的代价是你的所有物品、成就、等级都消失在虚空。

            不过理论上名字改回来就可以找回。

    **关于 HMCL 配置说明**
    - 主界面左下角`设置`
      - `全局游戏设置` > `游戏内存`
        对于大型整合包，建议不低于 12 GB，具体看你的内存配置 ~~(你应该能分清内存和硬盘大小，对吧)~~
      - `全局游戏设置` > `游戏 Java`
        只要你正确安装了 Java，应该能自动检测
      - `全局游戏设置` > `版本管理`
        一定要使用`各实例独自存在`，不然后面换版本玩你会很难受

## **第一步** 安装「齿轮盛宴」原整合包
1. 下载带有`Client`字样的客户端整合包 (大概70MB大小)
   - QQ 群中查看群文件，在 `⚙️齿轮盛宴整合包` 文件夹中
   - 下载名为 `[Client]Create-Delight-Remake-vXXX.zip` 的文件
   - 如果文件失效、没找到，提醒群管理 `Te amo`
    > 群主是 CB，问他没用
2. 如果你没有下载和配置过 `HMCL`，**先去配置** (第零步介绍配置方法)
3. 不用解压，压缩包拖到 `HMCL` 主界面中
   - 过程会自动安装缺失文件和模组
   - 由于从 Modrinth 等网站拉取，依赖网络环境
   - 总大小大概 77 MB -> 700 MB (首次游戏前) -> 1 GB (首次游戏后)

## **第二步** 下载并部署更新器
1. 下载 `Updater.exe` 可执行文件
   - 同样在 Q 群中同一个群文件夹
2. 找到指定文件夹
   - 如果你配置过 HMCL 的 `版本管理` 为 `各实例独自存在`，那么你的游戏文件夹在启动器所在文件夹下 `.minecraft/versions/Create-Delight-Remake`
3. 把 `Updater.exe` 放在指定文件夹
   ```text
    启动器所在文件夹
    ├─ .minecraft
    │   └─ versions
    │       └─ Create-Delight-Remake
    │           ├─ mods
    │           ├─ 其他文件
    │           └─ Updater.exe < 放在这个位置!!
    └─ HMCL.exe < 你的启动器在这
    ```
4. 每次有更新，你只需要双击这个`更新器`他会自动拉取并更新
   - 推荐: 第一次启动前先运行一次，因为你的包里只有「齿轮盛宴」原版内容
